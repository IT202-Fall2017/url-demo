URL examples

This page shows the images loaded for different source URL values.

https://developer.mozilla.org/en-US/docs/Learn/Common_questions/What_is_a_URL

Image src = image.jpg

Image src = ./image.jpg

Image src = ../image.jpg

Image src = images/image.jpg

Image src = /images/image.jpg

The initial slash makes this a "root relative" or "implicit domain" url; the browser constructs the URL starting at the domain root. Check the dev tools network tab to see the URL the browser created.

Image src = //image.jpg

This is an "implicit protocol URL. This image is at the top level of my folder, but it's not at the top level of the domain, so no image is loaded. Again, check the dev tools network tab.
